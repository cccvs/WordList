## 测试数据格式说明
- core_test：分为三个部分，每部分空行分隔
  - 输入
    - 第一行整数 n，后面 n 行为输入的 n 个 单词
    - n 不超过 30
    - 单词不重复、全小写、长度不超过 20
  - 参数：一共 5 个，含义和取值如下
    - mode：n、w、c（出现 n 时后面参数均为 -）
    - head：a-z，-（表示不选）
    - tail：取值同上
    - reject：取值同上
    - enable_cycle：r，-（表示不选）
  - 答案
    - 整数 m，表示最长长度
- exc_test：分为两部分，每部分占一行
  - 参数
    - 一个整数 n 表示参数数量
    - 后面 n 个字符串，表示命令行参数
  - 答案
    - 抛出的异常的内容
